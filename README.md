# WHA-HTTP

WHA-HTTP is an open-source, self-hosted WhatsApp Client that lets you connect multiple accounts and automate messaging workflows over HTTP, WebSockets, and webhooks.

## Contributing

Contributions are welcome. Please feel free to submit a pull request. Keep PRs focused and test with at least one real WhatsApp account before submitting.
