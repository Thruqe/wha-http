# WHA-HTTP Documentation

Welcome to the WHA-HTTP documentation.

WHA-HTTP is an open-source, self-hosted WhatsApp gateway that lets you connect multiple accounts and automate messaging workflows over HTTP, WebSockets, and webhooks.

## What's in this docs?

- **Getting Started** – Installation and setup
- **API Reference** – HTTP endpoints and WebSocket events
- **Webhooks** – Event delivery configuration
- **Authentication** – API keys and security
- **Troubleshooting** – Common issues and fixes
